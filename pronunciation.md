### Sound-spelling correspondences

| Sound     | Spelling  | Examples  |
| --------- | --------- | --------- |
| /ɪ/       | **i**     | th**i**s, l**i**sten |
|           | **y**     | g**y**m, t**y**pical |
|           | **ui**    | b**ui**ld, g**ui**tar |
|           | **e**     | pr**e**tty |
| /i:/      | **ee**    | gr**ee**n, sl**ee**p |
|           | **ie**    | n**ie**ce, bel**ie**ve |
|           | **ea**    | r**ea**d, t**ea**cher |
|           | **e**     | th**e**se, compl**e**te |
|           | **ey**    | k**ey**, mon**ey** |
|           | **ei**    | rec**ei**pt, rec**ei**ve |
|           | **i**     | pol**i**ce |
| /æ/       | **a**     | c**a**n, p**a**sta |
| /ɑ:/      | **a**     | c**a**n't, d**a**nce * |
|           | **ar**    | sc**ar**f, b**ar**gain |
|           | **al**    | h**al**f |
|           | **au**    | **au**nt, l**au**gh |
|           | **ea**    | h**ea**rt |
| /ʌ/       | **u**     | f**u**n, h**u**sband |
|           | **o**     | s**o**me, m**o**ther |

|           | **a**     | w**a**tch, w**a**nt |

\* In American English the sound in words like *can't* and *dance* is the shorter
**/æ/** sound, like *can* and *man*.
